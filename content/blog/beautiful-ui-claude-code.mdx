## Key takeaways

### 1. Default AI design is recognizable — and that's a problem

Chris builds a standard landing page for an AI agency with no design prompt. The result: functional but generic. Blue-purple palette, basic spacing, the usual AI aesthetic. It's not terrible, but it's instantly recognizable as AI-generated. The skill exists specifically to break past this ceiling.

### 2. The skill enforces real design principles

After installing and running the front-end design skill with zero additional context, the landing page transforms. Black, white, and one accent color. Proper typography hierarchy. Glow effects on scroll. Hover states. Fade-in animations. Consistent styling. A subtle grid pattern behind cards. These aren't random improvements — the skill encodes actual design principles that most AI outputs ignore.

### 3. Skill + screenshot reference is the power move

The second test adds a Dribbble screenshot as a design reference. The results shift dramatically: soft gradients and pastels, rounded corners, blur shadows, the Outfit font, smooth animations. Claude even narrates the transformation — "from brutalist to soft modern." Combining a skill with a visual reference gives the model both principles and direction. One without the other produces weaker results.

### 4. It's not perfect — human review is still mandatory

Even with the skill, some sections break. Emoji fallbacks instead of proper icons. Misaligned text. Formatting errors in certain card layouts. The call-to-action button didn't stand out against the dark background. Chris calls these out honestly. The skill gets you 80% of the way to professional quality — the last 20% still requires a designer's eye.

### 5. Design systems multiply the effect

Chris suggests the next level: create a design system framework in your codebase, then use the front-end skill to design within that framework. The skill provides the design instinct; the system provides the constraints. Together, they produce output that's closer to what a professional designer would ship.
